# REPOLENS — AI-Powered Repository Intelligence Platform

REPOLENS transforms unfamiliar GitHub repositories into instantly understandable engineering systems.

Drop any public repository URL and receive:

* AI-generated architectural breakdowns
* dependency intelligence
* technology stack detection
* key file analysis
* semantic repository understanding
* interactive visualization graphs

Built using React, Vite, Node.js, Groq LLM inference, and D3.js to accelerate developer onboarding and codebase exploration.
